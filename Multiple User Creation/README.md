#Multiple User Creation
#Problem
I was asked to add 10 users to Greenwood Accountants. 

##How I solve the problem
Creating multiple users one after the other can consume time. I have to bulk create the users using csv file.

##Key lessons
This task has reinforce something important in my learning

#How I will approach this henceforth

>If my manage say:
 "add multiple users" 
>I will break it down into:
1. Collect the details of the users
2. Check if it is in csv file, if not
3. Provision it into csv file
4. upload the file
5. And check if it has been added

#Conclusion

This felt like a very realistic issue — something that would easily come up in a real support environment.

More importantly, it helped me build the habit of **assuming the file is in csv**, but instead working through it step by step until the users are added.
